Welcome to the Switzerland Site

This is a demo of a country switcher that redirects you to the appropriate site based on your geographical location. For your project, I will use a geolocation API to determine the user's location and update the link accordingly.

For this demo, I have used Javascript to return a random choice between Germany and Switzerland. To see how it works, refresh the site several times and incase the computer has randomly chosen Germany, the pop up will appear. If not, you will stay on this sample Switzerland website
